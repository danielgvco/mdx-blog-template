# MDX Syntax Documentation

MDX is an authorable format that lets you seamlessly write JSX in your markdown documents.

## Headers

To create headers, you use the hash (#) symbol. The number of hashes corresponds to the header level.

Example:
```bash
# This is a level 1 header
## This is a level 2 header
### This is a level 3 header
```
Result:
# This is a level 1 header
## This is a level 2 header
### This is a level 3 header

## Emphasizing Text

For **bold** text, use two asterisks or underscores. 
Example:
```bash
**This is bold text**
```

For _italic_ text, use one asterisk or underscore. 
Example:
```bash
_This is italic text_
```

## Links

To create a hyperlink, use square brackets for the link text and parentheses for the URL. 

Example: 
```bash
[This is a link](https://example.com)
```
Result:

[This is a link](https://example.com)

## Lists

For unordered lists, use a hyphen (-), asterisk (*), or plus (+).
Example: 
```bash
- Item 1
- Item 2
```
Result:
- Item 1
- Item 2

For ordered lists, simply number each line.
Example: 
```bash
1. Item 1
2. Item 2
```
Result:
1. Item 1
2. Item 2

## Blockquotes

To create a blockquote, use the greater than symbol (>).
Example:
```bash
> This is a blockquote.
```
Result:
> This is a blockquote.

## Code

For inline code, use a single backtick (\`).
Example: \\\`This is inline code\\\`

For code blocks, use triple backticks (\```)

Example: 
```bash
\```bash
\This is a code block.
\```
```
Result:
```bash
This is a code block.
```

For syntax-highlighted code blocks, use triple backticks and specify the language.

Example: 
```bash
\```javascript
\console.log('This is a JavaScript code block.');
\```
```
Result:
```javascript
console.log('This is a JavaScript code block.');
```

## Tables

To create a table, use vertical bars and hyphens.

Example: 
```bash
| Column 1 | Column 2 | Column 3 |
| -------- | -------- | -------- |
| Row 1    | Data     | Data     |
| Row 2    | Data     | Data     |
```
Result:
| Column 1 | Column 2 | Column 3 |
| -------- | -------- | -------- |
| Row 1    | Data     | Data     |
| Row 2    | Data     | Data     |
